Cassette Future Magazine — a retro-futuristic publication built with Hugo. Cassette futurism aesthetic, custom theme, dispatches from year 2935.

## Development

```bash
git clone https://github.com/AnarchySC/magazine.git
cd magazine
hugo server --bind 0.0.0.0 --port 34405 --disableFastRender --noHTTPCache
```
